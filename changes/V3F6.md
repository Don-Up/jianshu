# Task 6: Draft Auto-save for Editor

## 第一部分: 概述

为文章编辑器添加草稿自动保存功能，防止编辑内容丢失。新建 `useDraft` Hook 管理 localStorage 草稿，在 ArticleEditor 中实现 30 秒自动保存、页面加载时恢复草稿、发布成功后清除草稿等功能。

## 第三部分: 详细改动说明

### use-draft.ts[新增](apps/web/src/hooks/use-draft.ts)

```typescript
export interface DraftData {
  title?: string;
  content?: string;
  excerpt?: string;
  tags?: string[];
  coverImage?: string;
}

export function useDraft(key: string): UseDraftResult {
  const saveDraft = useCallback((data: DraftData) => {
    localStorage.setItem(key, JSON.stringify(data));
  }, [key]);

  const loadDraft = useCallback((): DraftData | null => {
    const stored = localStorage.getItem(key);
    if (!stored) return null;
    try {
      return JSON.parse(stored) as DraftData;
    } catch {
      return null;
    }
  }, [key]);

  const clearDraft = useCallback(() => {
    localStorage.removeItem(key);
  }, [key]);

  const hasDraft = useCallback((): boolean => {
    return localStorage.getItem(key) !== null;
  }, [key]);

  return { saveDraft, loadDraft, clearDraft, hasDraft };
}
```

**说明：**
- `saveDraft` - 序列化草稿数据并存入 localStorage
- `loadDraft` - 从 localStorage 读取并解析草稿，解析失败返回 null
- `clearDraft` - 删除指定 key 的草稿
- `hasDraft` - 检查草稿是否存在

### article-editor.tsx[修改](apps/web/src/components/article/article-editor.tsx)

**1. 引入 useDraft Hook**
```typescript
import { useDraft } from '@/hooks/use-draft';
```

**2. 初始化草稿管理**
```typescript
const draftKey = slug ? `article-draft-${slug}` : 'article-draft-new';
const { saveDraft, loadDraft, clearDraft, hasDraft } = useDraft(draftKey);
```

**3. 自动保存 Effect（30秒间隔）**
```typescript
useEffect(() => {
  if (isEditing || slug) return;

  const intervalId = setInterval(() => {
    const currentData = { title, content, excerpt, tags, coverImage };
    if (title.trim() || content.trim()) {
      saveDraft(currentData);
      setLastSaved(new Date());
    }
  }, 30000);

  return () => clearInterval(intervalId);
}, [title, content, excerpt, tags, coverImage, isEditing, slug, saveDraft]);
```

**4. 草稿恢复 Effect（页面加载时）**
```typescript
useEffect(() => {
  if (!isEditing && !slug) {
    const savedDraft = loadDraft();
    if (savedDraft) {
      if (savedDraft.title && !initialData?.title) setTitle(savedDraft.title);
      if (savedDraft.content && !initialData?.content) setContent(savedDraft.content);
      toast.info('已恢复上次的草稿');
    }
  }
}, []);
```

**5. 发布成功后清除草稿**
```typescript
if (result.success && result.data) {
  clearDraft();
  toast.success(publish ? '发布成功' : '草稿保存成功');
  router.push(`/article/${result.data.slug}`);
}
```

## 其它

### 设计亮点

- **防丢失**：自动保存确保编辑内容不会因意外关闭浏览器而丢失
- **无干扰**：仅在新文章编辑时自动保存，编辑已有文章不保存（避免覆盖服务器内容）
- **用户知情**：显示最后保存时间和未保存草稿状态提示

### 注意事项

- localStorage 有 5-10MB 限制，草稿仅存文本不会超限
- 解析 localStorage 中的 JSON 失败时会静默返回 null，不影响用户使用
