| 方法 | 路径 | 说明 |
|------|------|------|

## 测试验证

### 启动服务

```bash
cd apps/api
pnpm start:dev
```

### 测试 API 端点

```bash
# 获取文章评论（无需认证）
curl http://localhost:4000/api/v1/articles/test-slug/comments

# 创建评论（需要 JWT Token）
curl -X POST http://localhost:4000/api/v1/articles/test-slug/comments \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"content": "测试评论"}'

# 点赞评论
curl -X POST http://localhost:4000/api/v1/comments/<commentId>/like \
  -H "Authorization: Bearer <token>"
```

## 设计亮点

1. **模块化架构**: CommentsModule 独立封装，便于维护和测试
2. **RESTful 路由**: 嵌套资源使用 `articles/:slug/comments`，单资源使用 `comments/:id`
3. **版本控制**: 所有端点遵循 `/api/v1/` 版本约定
